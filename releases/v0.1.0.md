# Autopraxis v0.1.0

Initial public release of Autopraxis: self-improving workflow skills for AI agents.

## Highlights

- 16 reusable skills with YAML frontmatter.
- Native Claude Code, Codex, and Me Write Code skills-plugin manifests.
- Portable installer for plugin-root, direct-skill, and markdown-bundle installs.
- Structured doc-authoring standards and templates for PRDs, DDs/RFCs, technical plans, ADRs, roadmaps, and RCAs.
- Visual documentation guidance using Mermaid and Graphviz/DOT diagrams.
- Backprop workflow for telemetry-driven improvement of prior workflow runs.

## Install

Clone the repo, then install as a native skills plugin:

```bash
node bin/autopraxis.mjs install --target claude-plugin
node bin/autopraxis.mjs install --target codex-plugin
node bin/autopraxis.mjs install --target mewrite-plugin
```

Direct skill and fallback bundle targets are also available:

```bash
node bin/autopraxis.mjs list-targets
```

## Validation

Validated for this release:

```bash
npm test
npm pack --dry-run
```

## Known Limitations

- npm publishing is disabled with `"private": true` while plugin install flows stabilize.
- Marketplace publishing is not automated yet.
- Workflow eval framework, additional workflow selection, and token/cost optimization are tracked in open issues.
